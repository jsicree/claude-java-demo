claude-java-demo

A Spring Boot REST API demonstrating hexagonal (Ports & Adapters) architecture. Domain logic is fully isolated from framework concerns; persistence is backed by MySQL via JPA.

Java 17 Spring Boot 4.0 Spring Data JPA MySQL 8.4 Docker
Swagger UI OpenAPI JSON

API Endpoints

Products

MethodPathDescription
POST /api/products Create a new product
GET /api/products List all products
GET /api/products/{id} Get a product by ID

Customers

MethodPathDescription
POST /api/customers Register a new customer
GET /api/customers List all customers
GET /api/customers/{id} Get a customer by ID

Architecture

  • domain/ — Pure Java entities and domain exceptions; zero framework dependencies
  • application/port/in/ — Use-case interfaces (CreateProductUseCase, RegisterCustomerUseCase, …)
  • application/port/out/ — Repository port interfaces (ProductRepository, CustomerRepository)
  • application/service/ — Package-private use-case implementations
  • adapter/in/web/ — REST controllers and request/response records
  • adapter/out/persistence/ — JPA entities and Spring Data repository adapters
